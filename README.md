# Quick Start
## Chrome Extension 
```
$pnpm install
$pnpm dev
```
and load the unpacked Chrome extension from /dist directory in Chrome

# Test Scenarios
```
$pnpm serve-tests
```
and navigate to http://127.0.0.1:8080 and select a test payload
